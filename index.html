Gumption is a digital initiative tracking system for tabletop roleplaying game. It assists the game master in any combat situation so he or she can easily manage the flow of the game. Tools in Gumption will make it easy to create all the participants in combat, reorder them on the fly, add new participants if necessar, and save ongoing campaign characters for quick use at any time.

Combat systems in tabletop role playing games generally involve some sort of initiative tracking system to determine when characters get to act during a given turn. Tracking initiative is a task that is easy to mess up when things get sticky. Dry erase boards, magnetic slates, numbering systems; all of these systems can easily get forgotten about in the middle of combat. Coming back to the initiative order after a longer turn can be more time consuming than it should be. Gumption removes the manual process of tracking initiative, makes it easy to forget about systems and lets you just play the game.

FAQ

What is a tabletop role-playing game?
A tabletop role-playing game is a game in which the players take the role of characters they have created and play through imaginary scenarios. The game is moderated by one of the players, generally referred to as the game master, who takes the role of any characters not controlled by the players. The game has a set of rules with the players use to determine the success or failure of their characters within the game.
Why do I need a tool to track initiative?
Initiative is one of those things in a role-playing game that is very easy to forget about. During combat, the game master has other things to think about like enemies for the players to fight, rules to handle the combat, and all sorts of other things. With all of those balls in the air it is a very common problem to run into.
What role-playing games is Gumption compatible with?
Gumption was built for role-playing games that feature a lot of combat, but it wasn't designed with any given system in mind. It is simple enough that it can be applied to any system in which turn order during combat is tracked with initiative of some sort.
